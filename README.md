# 🗳 Online Voting System

A full-stack web application that allows users to register, cast votes securely, and view real-time results.

---

## 🚀 Tech Stack

* **Backend:** Java (Spring Boot)
* **Frontend:** HTML, CSS, JavaScript
* **Architecture:** REST API based

---

## ✨ Features

* 🧾 Voter Registration System
* 🔒 One Vote Per User (Prevents Duplicate Voting)
* 📊 Real-Time Vote Counting
* 🧑‍💼 Candidate Management
* 🌐 RESTful API Integration

---

## 🖥️ Project Structure

```bash
voting/
│
├── src/                  # Spring Boot Backend
├── frontend/             # HTML, CSS, JS UI
├── pom.xml               # Maven Dependencies
```

---

## ▶️ How to Run

### 🔹 Backend

```bash
./mvnw spring-boot:run
```

Server runs at:

```
http://localhost:8080
```

---

### 🔹 Frontend

Open in browser (recommended with Live Server):

```
frontend/index.html
```

---

## 📡 API Endpoints

| Method | Endpoint        | Description    |
| ------ | --------------- | -------------- |
| POST   | /api/register   | Register voter |
| GET    | /api/candidates | Get candidates |
| POST   | /api/vote       | Cast vote      |
| GET    | /api/results    | View results   |
